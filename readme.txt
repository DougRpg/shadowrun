Tradução Shadowrun - 2016 - DougRPG - Versão Beta 1 - 12/12/2016 03:37am
------------------------------------------------------
Esta é a tradução do jogo Shadowrun de Super Nintendo para o português.
Esta é a minha primeira tradução lançada. Esta versão ainda é Beta, porém 100% jogável.




Como aplicar o patch
------------------------------------------------------
É necessário usar o Rom "Shadowrun (U) [!]" com CRC32 "3F34DFF0".


Existem duas formas de aplicar o patch, usando o IPS ou o BPS.

O IPS é a forma mais comum, porém menos segura, pois não checa se a Rom que você está usando é a correta. Pra usar o IPS deve-se usar qualquer ferramenta de IPS, como o LunarIPS:
http://www.romhacking.net/utilities/240/

O BPS é bem mais seguro, pois garante que a Rom é a correta e evita problemas na hora de jogar. É necessário usar a ferramente Beat pra aplicar o BPS:
http://www.romhacking.net/utilities/893/

Veja a seção de Compatibilidade mais abaixo pra saber quais os emuladores compatíveis.




História da tradução
------------------------------------------------------
Eu comecei essa tradução em meados de 2015. Anteriormente já tinha analisado a Rom, lá pra 2013, porém tinha achado o jogo muito complicado pois os ponteiros eram todos espalhados pela Rom, o que dificultava muito o processo de tradução.
No romhacking.net existe uma ferramenta que extrai todos os textos do jogo, feita por um cara chamado CUE, porém como existe esse problema dos ponteiros eu não consegui avançar. Fora isso existia outros problemas, como fonte comprimida, menus pra reorganizar, etc.
Porém em 2015, depois de debugar vários jogos eu ganhei a experiência necessária pra traduzir este jogo. Consegui uma solução muito simples para este problema dos ponteiros e consegui alterar a fonte e todos os menus. 
A partir daí comecei a traduzir os textos e fazendo o romhacking necessário. No começo tentei encontrar alguém pra me ajudar na tradução dos textos, mas não encontrei, então traduzi eu mesmo todos os textos.
Li vários sites a respeito do Shadowrun pra tentar manter todos os termos e gírias usadas no universo do Shadowrun. Esse jogo tem uma temática cyberpunk e mais adulta, então tentei manter o espírito do universo nos textos traduzidos. 
Agora a tradução deste jogo é uma realidade.




Status da tradução
------------------------------------------------------
Esta é uma versão Beta, porém 100% jogável. Devido a falta de tempo não consegui lançar uma versão 100% final e precisarei de ajuda para dar o polimento final, retirando todos os bugs e erros de gramática que por ventura ainda ficaram.
Depois que eu finalizei a tradução dos textos e todo o romhacking eu já joguei esta tradução uma vez do começo ao fim, arrumando tudo o que encontrei pelo caminho. Portanto nesta primeira passada já arrumei todos os problemas que encontrei, então provavelmente poucos erros (ou nenhum) aparecerão pra quem for jogar. Mas com certeza ainda deve ter algum errinho pra arrumar, pois nesta jogada eu não fiz tudo o que podia ser feito.

Abaixo está uma lista do que já foi feito:

- 100% da tradução dos diálogos.
- Romhacking principal feito.
- Introdução traduzida
- Primeira jogada de teste feita, corrigindo todos os erros encontrados.

O que ainda falta ser feito:

- Mais testes para tirar possíveis erros.
- Os créditos não foram traduzidos ainda, mas não influencia nada no jogo. E o final não tem texto não traduzido, então só preciso traduzir aqueles "directed by", etc, o que não interfere em nada.
- A tela de opções ainda está em inglês. Só traduzi até o momento a primeira tela de escolha, que tem os textos "Começar novo jogo", etc. Mas não interfere muita coisa pra quem for jogar.
- A tela de login da Matrix ainda está em inglês (quando você loga em algum computador), mas são poucas palavras que não são necessárias para o andamento do jogo. Eu até estou pensando em deixar em inglês mesmo pois acho que até fica legal a Matrix com alguma palavras em inglês. Você é um hacker que está hackeando a Matrix, então se vira pra aprender a hackear.
- Eu coloquei uma tela inicial na hora que o jogo inicia e tentei colocar uma música cantada, porém ainda não consegui. Fica pra próxima.
- Preciso dar uma limpeza geral na Rom e uma reduzida no tamanho. No começo eu expandi o jogo pra 6M e fui colocando um monte de lixo, então tenho que dar uma limpada. Mas não interfere em nada no jogo, é só uma organização interna. 

Resumindo, a tradução está 99%.

Para reportar erros mande email para "dougrpgromhacking@gmail.com".




Compatibilidade
------------------------------------------------------
Eu fiz todo o debug e testes utilizando o Bsnes, portanto a compatibilidade com o hardware é garantida. Mas pra jogar no Bsnes/Higan provavelmente você terá que adaptar o manifest pra poder carregar o formato ExLoRom. Se você não sabe fazer isso então use o Snes9x ou Zsnes.

Joguei também usando o Snes9x 1.53 e é 100% compatível.

Também é possível jogar no Zsnes, porém é necessário usar a versão compatível com 8MB. É essa daqui:
http://www.romhacking.net/utilities/904/

Não testei em outros emuladores, como emulador de Psp, etc... Porém se o emulador não rodar é porque não suporta jogos maiores de 4MB, então é culpa do emulador, não do jogo.

Lembre-se de que é uma versão Beta, então não liberei com a missão de rodar em todos os emuladores existentes. Roda nos principais.
